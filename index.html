How to Share Software

This website is about the different ways to share software source code. Shared source code is called "source available," and there are four levels of source available software.

Source Available

Most Available

Level 4: Public Domain Software

Public domain does not exist in all countries, but where it does it is the highest level of sharing. There are no restrictions.

Level 3: Free and Open Source Software (FOSS) Projects

Open Source software is shared with minimal restrictions, such as no use without attribution, no use without sharing-alike, or no patent litigation. It is the best fit for community projects.

Level 2: Open Products

Some companies make their source code publicly available with usage restrictions in line with their business model, such as: no use in a competing product, or no redistribution.

Level 1: Privately Shared Software

Some companies make their source code available to customers or other parties under contractual license or confidentiality terms.

Least Available

Closed Source

Source code is not shared outside of the company that owns it.
